# Ask Arpita

A personal AI assistant web app built with HTML, CSS, JavaScript, Node.js, Express and Gemini API.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Create `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

3. Start the server:

```bash
node server.js
```

4. Open:

```text
http://localhost:3000
```

## Features

- Landing page
- AI chat interface
- Gemini API backend
- Saved questions using localStorage
- Voice input using browser speech recognition
- Tools and settings sections
